Q: To view what's written in a file.
A: cat filename

Q: To change the access permissions of files.
A: chmod permissions filename

Q: To check which commands you have run till now.
A: history

Q: To remove a directory/ Folder.
A: rm -r directoryname

Q: To create a fruits.txt file and to view the content.
A: echo "" > fruits.txt
   cat fruits.txt

Q: Add content in devops.txt (One in each line) - Apple, Mango, Banana, Cherry, Kiwi, Orange, Guava.
A: echo -e "Apple\nMango\nBanana\nCherry\nKiwi\nOrange\nGuava" > devops.txt

Q: To Show only top three fruits from the file.
A: head -n 3 devops.txt

Q: To Show only bottom three fruits from the file.
A: tail -n 3 devops.txt

Q: To create another file Colors.txt and to view the content.
A: echo "" > Colors.txt
   cat Colors.txt

Q: Add content in Colors.txt (One in each line) - Red, Pink, White, Black, Blue, Orange, Purple, Grey.
A: echo -e "Red\nPink\nWhite\nBlack\nBlue\nOrange\nPurple\nGrey" > Colors.txt

Q: To find the difference between fruits.txt and Colors.txt file.
A: diff fruits.txt Colors.txt
